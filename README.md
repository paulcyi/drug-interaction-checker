# 💊 CheckMyMeds

**CheckMyMeds** is a modern, AI-powered medication safety platform designed to help users quickly identify potential drug interactions, manage personal medication profiles, and receive personalized health insights. Built for families, caregivers, veterans, and anyone taking multiple medications, CheckMyMeds aims to make med safety clear, accessible, and intelligent.

---

## 🚀 Current Features

- ✅ Simple, clean UI built with Streamlit
- ✅ Basic drug interaction checker using mocked data (DrugBank API coming)
- ✅ Domain secured: [checkmymeds.app](https://checkmymeds.app)
- ✅ Designed for eventual iOS + Android native apps

---

## 🧠 Upcoming AI Features

- 🔍 AI-generated explanations for drug interactions in plain English
- 🔠 Fuzzy matching for typos and brand/generic names
- 📊 Risk scoring based on medication class, age, or conditions
- 🧬 NLP summarization of side effects, risks, and labels

## 📦 Roadmap

### ✅ MVP Goals (Next 1–2 Months)
- Deploy live to Streamlit Cloud at `checkmymeds.app`
- Add DrugBank API integration with fallback to mock data
- Add login + saved profiles
- Limit free tier to 5 checks/day
- Add disclaimer + analytics

### 🧪 Premium Tier Goals (3–4 Months)
- Unlimited checks
- AI-powered explainers
- Med reminders + caregiver alerts
- Shared family plans + history logs

### 🔬 Long-Term
- NLP-based drug name interpreter
- Risk prediction models
- Mobile app (native iOS + Android)
- Partnerships with pharmacies or telehealth platforms
